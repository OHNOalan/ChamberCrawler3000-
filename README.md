CC3K is a C++ terminal game.

Instruction:
1. choose your choose your character
2. choose the difficulty level
3. Action

Action:
1. Move (no/so/we/ea)
2. Attack (a + (no/so/we/ea))
3. Drink Potion (u + (no/so/we/ea))
4. Restart (r)
5. Quit (q)

GameObject:
@: Player
G: Treasure
P: Potion
C: Compass
\\: Stairs
.: Tile
B: BarrierSuit
#: Passage
+: Door
|: Vertical Wall
-: Horizontal Wall
W/V/N/M/D/X/T: Enemies
